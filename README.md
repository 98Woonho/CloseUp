## ▶️ 개발 동기
#### 일상에서 정보 부족으로 인해 구하기 힘든 서비스들을 여러 전문가들이 모여 저렴한 가격에 쉽게 이용할 수 있는 사이트를 통해, 고객의 편의성과 접근성을 높이고자 이 사이트를 개발하게 되었습니다.
